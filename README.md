# Автоматизированный EDA анализатор

Веб-приложение для автоматического разведочного анализа данных (EDA) с возможностью загрузки любых CSV файлов.

## Запуск приложения

### Предварительные требования
- Python 3.8+
- Виртуальное окружение с установленными зависимостями

### Установка зависимостей
```bash
# Активация виртуального окружения
.venv\Scripts\activate

# Установка пакетов
pip install -r requirements.txt
```

### Запуск приложения
```bash
# Из корневой директории проекта
streamlit run app.py
```

Приложение будет доступно по адресу: `http://localhost:8501`

## Возможности анализа

### 1. Основная информация о датасете
- Размер датасета (строки × столбцы)
- Типы данных каждого столбца
- Количество пропущенных значений

### 2. Анализ пропущенных значений
- Тепловая карта пропусков
- Гистограмма количества пропусков по признакам
- Подробная информация о пропусках

### 3. Анализ числовых признаков
- Описательные статистики (среднее, медиана, квартили, мин/макс)
- Гистограммы распределения (для отдельных признаков и всех сразу)
- Boxplots для выявления выбросов (для отдельных признаков и всех сразу)

### 4. Анализ категориальных признаков
- Количество уникальных значений
- Наиболее и наименее частые значения
- Визуализация распределения (bar charts)

### 5. Анализ выбросов
- Метод IQR (межквартильный размах)
- Количество выбросов по каждому признаку
- Границы нормальных значений
- Просмотр строк с выбросами

### 6. Корреляционный анализ
- Матрица корреляций для числовых признаков
- Тепловая карта корреляций
- Scatter plots для пар признаков

### 7. Сравнение распределений
- Boxplot сравнение числовых признаков по категориям
- Наложенные гистограммы для разных категорий

### 8. Сводные таблицы
- Средние значения по группам
- Минимальные и максимальные значения
- Количество наблюдений в каждой группе

## Использование

1. Откройте приложение в браузере
2. Нажмите "Выберите CSV файл" и загрузите ваш датасет
3. Исследуйте различные разделы анализа:
   - Изучите общую информацию о данных
   - Проанализируйте пропуски и выбросы
   - Посмотрите распределения признаков
   - Изучите взаимосвязи между признаками

## Требования к данным

- Формат: CSV

Приложение автоматически определит типы данных и проведет соответствующий анализ.
